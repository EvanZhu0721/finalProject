# Typing Lane 1.0.1 - 游戏机制

版本：1.0.1

## 核心循环

玩家在两条平行 lane 上生存、换路、攻击、收集 XP。

- 画面下方显示两条 lane 的大号目标单词。
- 完整输入上路单词：进入上路并攻击上路目标。
- 完整输入下路单词：进入下路并攻击下路目标。
- 换路时玩家角色使用非线性平滑动画移动到新 lane。
- 目标单词完成后立刻刷新。
- 正确输入会点亮前缀并触发输入脉冲；完整单词会触发完成扩散特效。
- 字符失配会清空当前前缀，并用新字符重新判定两条 lane。
- Space 在升级就绪时打开升级菜单；未就绪时不触发额外动作。
- F2 切换英文 / 中文 UI；F11 切换全屏。

## 玩家状态

- 初始 HP：36。
- 早期普通碰撞约 6 次容错。
- 分数来源：击败目标、收集 XP。
- Combo 来源：连续完成 lane 单词。
- 当前 WPM、分数、HP、压力、升级进度显示在 HUD 中。
- High score 记录当前程序会话。

## 战斗

- 完成单词后发射蓝白基础子弹，命中该 lane 最近目标。
- 高稀有“自动炮”：无需打字也会连续攻击当前 lane；完成词会让自动炮短暂进入强化弹窗口。
- 高稀有“贯穿轨道”：完成词额外发射青色贯穿弹，沿当前 lane 命中多个屏内目标，每次穿透后伤害衰减 20%。
- 所有攻击都通过可见子弹实体移动和碰撞造成伤害。
- 敌人抵达玩家所在 lane 时，按敌人当前 HP 扣除玩家 HP。

## 敌人

目标类型：

- NORMAL：基础敌人。
- FAST：高速低 HP 敌人。
- TANK：高 HP 慢速敌人。
- SHIFT：lane 压力敌人。
- CACHE：高价值 XP 缓存目标。
- BOSS：阶段压力目标。
- 敌人直接从可见轨道右端出现，不再从屏幕外进入。

Boss 会在击破数达到门槛后出现，并带来更高 HP、得分和 XP 奖励。

## XP 与升级

普通和精英目标被击败后掉落 lane 绑定 XP 球。

- XP 球靠近玩家同 lane 时产生小范围吸附，吸附触发后会持续朝玩家当前位置移动。
- 玩家进入对应 lane 并靠近 XP 球即可收集。
- XP 球显示 `XP` 和数值。
- NORMAL：1 XP。
- FAST：2 XP。
- TANK / SHIFT / CACHE：4 XP。
- Boss：直接奖励大量 XP，数值为 `24 + BossLevel * 8`。

XP 进度满后生成一次待选升级。待选升级可累积，玩家按 Space 主动打开 3 卡升级菜单。

前五次升级阈值：`6 / 8 / 12 / 16 / 22 XP`。第六次起接入 `30 / 40 / 50...` 的中后期曲线。

当前升级组合：

- 常规：2 Common + 1 Uncommon。
- 进阶：1 Common + 2 Uncommon。
- 高阶：1 Common + 1 Uncommon + 1 High。
- 每 5 个升级等级提高 High 槽出现率。

当前卡牌框架：

- Common：HP 回复、HP 上限、伤害百分比、扳机调校、Combo 调律、词形、连段、错误修正、XP 牵引。
- Uncommon：换路相位、Boss / 精英伤害、f/t/k XP crossfeed、护栏、标记、反击、危险距离控制。
- High：自动炮、贯穿轨道。

扳机调校：

- 每级多发射 1 颗弹丸；总伤害提高 20%，并平分到所有弹丸。

第一组词形升级：

- 长词回报：完成 6-7 字母词修补少量 HP。
- 短词快枪：完成 4 字母词追加高速低伤子弹。
- 尾字爆点：完成词追加爆点弹，命中后小范围溅射。
- 元音收束：元音结尾词牵引同 lane 最近 XP 球。
- 硬辅音破甲：以 `t/k/p/d/g` 结尾的词对厚血目标追加伤害。
- 源码开关：`ENABLE_GROUP_ONE_UPGRADES`。

第二组 lane 连段升级：

- 同路压制：连续完成同一 lane 会增伤并减速前排目标。
- 上下换拍：连续完成不同 lane 会向双 lane 追加分路轻弹。
- 单路堡垒：同 lane 连续 4 词生成该 lane 护栏。
- 交替护栏：上下交替连续 4 词生成双 lane 护栏。
- 连击校准器：Combo 达到 20/40/60... 时追加精准弹。

第三组首字母与前缀升级：

- 首字母锁定：首字母明确锁定一条 lane 时追加伤害。
- 准确起笔：前三字母无回删完成时追加伤害。
- 前缀照明：半词前缀无回删时标记前排目标并增伤。
- 首字母税票：连续 3 个同首字母完成词生成额外 XP。
- 双前缀读屏：战术首字母组触发另一 lane 扫射。

第四组错误与 Backspace 升级：

- 干净收尾：无错且无回删完成词追加轻弹。
- 断错复位：失配后快速完成词追加复位爆点弹。
- 回删修正：Backspace 修正后完成词追加伤害。
- 错后冷静：失配后连续干净完成 2 词获得冷静守护。
- 回删反击：Backspace 修正后完成词追加反击爆点弹。

第五组 XP 与危险距离升级：

- 精准拾取：完成词时强化同 lane XP 牵引。
- 尾字牵引：元音结尾词牵引 XP 并减速前排。
- 长词专注：干净完成 6-7 字母词后获得短暂增伤。
- 近险成词：前排接近时完成对应 lane 词会击退前排。
- 压力阀：lane 有近身压力时完成词会减速整条 lane。
- 源码开关：`ENABLE_GROUP_TWO_TO_FIVE_UPGRADES`。

测试后台：

- 输入 `sudo` 打开测试后台。
- Up / Down 浏览升级，Left / Right 翻页。
- Enter 或 Space 授予当前升级。
- Esc 关闭后台。
- 后台列表同步普通升级池，并额外提供测试工具项。
- 后台工具项包含测试无敌和大量 XP。

## Boss 奖励

所有 Boss 使用统一奖励管线：

- 击破 Boss 获得大量直接 XP。
- XP 进入通用升级进度。
- 升级菜单由玩家按 Space 主动打开。
- Boss 等级会提高压力、HP 曲线和奖励数值。

## 压力成长

压力随时间、击破数和 Boss 等级提高：

```text
pressure = min(14, tick / 430 + kills / 9 + bossLevel * 2)
```

压力影响：

- 敌人移动速度。
- 敌人 HP 上限。
- FAST、TANK、SHIFT 的出现占比。
- Boss HP 成长。

难度选择仅调整敌人移动速度：

- Easy：目标速度 25 WPM，敌人移动较慢。
- Normal：目标速度 40 WPM，敌人基准速度。
- Hard：目标速度 60 WPM，敌人移动较快。

## 操作

- Enter：开始游戏；Game Over 后重开。
- 开始前 1 / 2 / 3：选择 Easy / Normal / Hard。
- 字母键：输入 lane 单词。
- Backspace：退回一个已输入字符。
- Space：打开待选升级。
- Esc：暂停 / 继续。
- F2：切换英文 / 中文 UI。
- F5：暂停时重开。
- F11：切换全屏。
- 升级菜单：Left / Right + Enter，或 1 / 2 / 3。
- 测试后台：输入 `sudo`。

## 显示

- 渲染使用 1920 x 1080 逻辑画布，并缩放到窗口或全屏。
- 渲染计时器约 60 FPS；游戏逻辑保持 33ms 固定步长，并通过插值绘制移动目标。
- 霓虹 UI 统一使用深色背景、青色 lane、金色升级提示和高亮输入反馈。
- 底部大号输入槽承担主要打字信息。
- 玩家换路、目标减速、前缀标记、护栏和 XP 牵引都有可见霓虹反馈。
- 升级就绪通过 XP icon、进度条、Space 键帽和脉冲提示显示。
- HUD、菜单、卡牌和战斗提示支持英文 / 中文。
